# Research Portfolio | Riccardo Gozzo
Ph.D. Student in Computational Finance at Scuola Normale Superiore, specializing in hybrid Deep Learning architectures for high-frequency market dynamics.

## Core Research & Projects
**Latent GNNs for Volatility Forecasting**: Development of a Latent Soft-Sparse GNN-HAR framework utilizing endogenous graph construction and gated graph convolutions to capture non-linear volatility spillovers.
**End-to-End Portfolio Optimization**: Implementation of TSMixer and LSTM architectures using Adaptive Sharpness-Aware Minimization (ASAM) and a custom Linear Utility Loss to stabilize weight allocation under non-stationary regimes.
**Rare-Event Pricing**: Development of the [amsSim] R package (CRAN) implementing Adaptive Multilevel Splitting methods for derivative pricing.

## Technical Profile
**Scientific Computing**: Python (PyTorch, TensorFlow), C++, R, MySQL.
